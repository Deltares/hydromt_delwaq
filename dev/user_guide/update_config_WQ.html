Updating a modelÂ¶

This plugin allows to update any components from a D-Water Quality model. To do so, list the methods to update in a configuration file. If needed edit your data library with new data sources required for the update and use the command:

activate hydromt-delwaq
hydromt update delwaq path/to/model_to_update -o path/to/updated_model -i delwaq_update.ini -d data_sources.yml -vv
