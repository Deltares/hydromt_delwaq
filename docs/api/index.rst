===
API
===

.. _api_model:

The HydroMT-DELWAQ API provides classes and methods to build, update, and analyze
DELWAQ models. The API is divided into the following sections:

- :ref:`D-Emission model <api_demissionmodel>`: Class for building and managing D-Emission models.
- :ref:`D-Water Quality model <api_delwaqmodel>`: Class for building and managing DELWAQ fate and transport models.
- :ref:`Advanced API <api_reference_advanced>`: In-depth information, scientific motivations and advanced usage.
